FAQs

Three steps to success...
  1. Login as a new or existing user
  2. Customise book ratings in the dashboard
  3. View your personalised recommendations

How can I log in as an existing user?

  • Head to the PageFlix home page, by selecting the logo in the navigation bar or by visiting the '/' route
  • Enter a user id in the input box, or select the refresh icon to generate a random valid user id
  • If the box is glowing green, this indicates that the user id is valid. However, if it glows red, this is an invalid id.
  • Once the box is glowing green, click the submit button to login as the specified user

How can I sign up as a new user?

  • Head to the PageFlix sign up page, by selecting the 'Sign Up' link in the navigation bar
  • A valid new user id is already provided in the grey box
  • To continue as this user, click the continue button
  • Ta Da! This new user now exists and is ready to provide recommendations

Where can I submit ratings and see my recommendations?

  • After logging in, or signing up, you will be redirected to your dashboard
  • Here, you can update and view your ratings
  • Scrolling to the bottom of the page, you will also be able to view your recommendations
  • There is also a pie chart, showing the breakdown of genres in your recommendations!

How can view my ratings?

  • Your previous ratings are visible in the dashboard (top right corner under 'All Ratings')
  • Here, you can view the score you provided for each rated title
  • You can also delete these ratings by selecting the trash can

How can I submit new ratings?

  • You can add ratings in the dashboard, under 'Update Rating'
  • Select a title from the list of titles, or click the refresh icon to randomly select an unrated title
  • Enter a rating into the input box (between 1 and 5 inclusive)
  • Press the up arrow icon to submit this rating
  • This rating should now appear to the right, in the list of all ratings

How can I update an old rating?

  • You can update ratings in the dashboard
  • Select a rating from the list of all ratings, or select a title from the list of titles
  • This will open the book in the update rating window
  • You can then edit this rating by entering a new value, and submit the rating by pressing the up arrow icon
  • The updated rating should now appear to the right, in the list of all ratings!

Can I delete a rating?

  • In your dashboard, view the list of previously rated titles
  • Select the trash can, next to the rating you wish to delete
  • Voila! Your old rating is gone!

How can I change the number of recommendations visible to me?

  • In your dashboard, scroll to the bottom of the page to the list of recommended titles
  • Select the toggle button (number of recommendations)
  • This will toggle the number of recommendations between 5 and 10
  • This will also update the genre breakdown accordingly!

What happens if I have not rated any titles?

  • If no ratings have been provided, the titles with the highest average ratings are recommended
